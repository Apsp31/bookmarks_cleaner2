Create `src/dedup.js` with three exported functions:

1. `normalizeUrl(rawUrl)` — Per D-03, implement all 7 normalization rules using WHATWG `URL` API:
   - `import { URL } from 'node:url'`
   - Define `TRACKING_PARAMS = new Set(['utm_source', 'utm_medium', 'utm_campaign', 'utm_term', 'utm_content', 'fbclid', 'gclid'])`
   - Wrap `new URL(rawUrl)` in try/catch; return `rawUrl` unchanged on parse failure (handles `javascript:`, `mailto:`, etc.)
   - Rule 1: `parsed.hash = ''`
   - Rule 2: `if (parsed.protocol === 'http:') parsed.protocol = 'https:'`
   - Rule 3: `if (parsed.hostname.startsWith('www.')) parsed.hostname = parsed.hostname.slice(4)`
   - Rule 4: `if (parsed.pathname !== '/' && parsed.pathname.endsWith('/')) parsed.pathname = parsed.pathname.slice(0, -1)`
   - Rules 5-7: `for (const key of [...parsed.searchParams.keys()]) { if (TRACKING_PARAMS.has(key)) parsed.searchParams.delete(key) }`
   - CRITICAL: Spread keys before iterating (`[...parsed.searchParams.keys()]`) to avoid live iterator mutation bug (Pitfall 1 from RESEARCH.md)
   - Return `parsed.toString()`

2. `dedupTree(node, seen = new Set())` — Per D-02, depth-first top-down walk:
   - If `node.type === 'link'`: normalize URL, check `seen` Set, return `null` if duplicate, else add to `seen` and return shallow copy `{ ...node }`
   - If folder: recurse children with `.map(child => dedupTree(child, seen)).filter(Boolean)`, return `{ ...node, children: newChildren }`
   - The `seen` Set is shared across all recursion levels (passed by reference)
   - Never mutate the original tree (per D-07/D-11)

3. `countLinks(node)` — Recursive count of all nodes where `type === 'link'`. Used by cleanup route for stats.

  <implementation>
Create `src/fuzzy.js` with exported functions:

1. Internal `isSimilarName(a, b)`:
   - `import { distance } from 'fastest-levenshtein'`
   - `const maxLen = Math.max(a.length, b.length); if (maxLen === 0) return false;`
   - `return distance(a.toLowerCase(), b.toLowerCase()) / maxLen <= 0.25` (per D-06, fixed 25% threshold)

2. Internal `collectFolders(node, result = [])`:
   - Recurse through tree collecting all folder nodes
   - SKIP nodes where `node.title === 'root'` (Pitfall 4 from RESEARCH.md)
   - Return flat array of folder nodes

3. `findMergeCandidates(tree)`:
   - Collect folders, compare all unique pairs O(n^2) — fine for <200 folders
   - Return array of `{ aId, aName, bId, bName, score }` where score is the distance ratio

4. `fingerprintSubtree(folder)`:
   - `import { createHash } from 'node:crypto'`
   - `import { normalizeUrl } from './dedup.js'`
   - Extract direct link children URLs, normalize each, sort, SHA-256 hash the JSON stringified array
   - Return hex digest string

5. `findDuplicateSubtrees(tree)`:
   - Collect folders, skip those with zero link children
   - Group by fingerprint using a Map
   - For groups with >1 entry: first is `keep`, rest are `remove`
   - Return array of `{ keepId, keepName, removeId, removeName }`

Create `test/fuzzy.test.js` — all tests BEFORE implementation (RED phase).
